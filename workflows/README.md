# Workflows directory

Install workflow definitions into this directory.

The base `pi-multiagent` skill intentionally ships without domain-specific workflows. Workflow definitions are provided by separate workflow packs, for example:

```text
workflow/pi-java-multiagent/workflows/
```

After installing a workflow pack, this directory should contain files such as:

```text
full-development.md
workflow-index.md
```

All installed workflows must enforce English-only generated documentation, comments, reports, and user-facing artifact text.
